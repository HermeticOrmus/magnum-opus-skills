# Using this repo with Cursor

The Magnum Opus is a transformation framework, not a code-generation pattern. The Cursor rule loads the stage vocabulary so the AI can reason in those terms.

Copy `.cursor/rules/magnum-opus.mdc` to your project's `.cursor/rules/` or `CLAUDE.md` to the project root.

`alwaysApply: false` — invoke when transformation is the work, not as constant background.
